S4 (Structured State Spaces for Sequence Modeling) is a deep sequence model built on a linear state-space model (SSM) that maintains a continuous hidden state and updates it recurrently, giving a form of compressed long-term memory of the sequence [@gu2022]. Each layer parameterizes a linear ODE/recurrence (state update x' = Ax + Bu, output y = Cx) whose hidden state summarizes all past inputs; the key technical contribution is a structured (diagonal-plus-low-rank) parameterization of the state matrix, initialized with HiPPO theory so the state optimally compresses history, and an efficient computation that lets the model run as a global convolution during training and as a recurrence at inference. This makes it strong on tasks requiring very long-range dependencies (e.g. the Long Range Arena), where attention struggles, while scaling sub-quadratically in sequence length. As "memory," the fixed-size state continuously absorbs and decays past information rather than storing it explicitly, so it cannot perform exact content-based recall of arbitrary earlier tokens the way attention or a kNN memory can. Its limitations on selective copying and in-context recall motivated input-dependent SSMs such as Mamba.
