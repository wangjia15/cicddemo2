# cicddemo

一个演示 CI/CD 流程的 Python 计算器库。

## 安装

```bash
pip install -e .
```

## 快速开始

```python
from calculator.core import add, subtract, multiply, divide, power

print(add(1, 2))       # 3
print(subtract(5, 3))  # 2
print(multiply(2, 4))  # 8
print(divide(10, 2))   # 5.0
print(power(2, 3))     # 8
```
